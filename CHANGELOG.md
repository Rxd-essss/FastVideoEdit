## [1.0.0] — 2026-06-12

Первый публичный релиз: локальный пайплайн монтажа talking-head видео для
YouTube — CLI + веб-редактор, вся обработка на вашей машине.

### Транскрипция и детекторы
- Транскрипция faster-whisper (large-v3, CUDA, int8_float16) с кэшем по хэшу
  входа и авто-фолбэком large-v3 → medium → small → CPU.
- Детекторы: паузы (тишина между словами), слова-паразиты (`fillers_ru.yaml`),
  мат по корням с allow-листом (`profanity_ru.yaml`).
- Акустический детектор хезитаций (VAD по волне): тянутые «э-э-э», бубнёж и
  микрообрывы, которые текстовые детекторы не видят.
- Подсказки неудачных дублей (фальстарты, повторы) через локальный LLM
  (Ollama, qwen3:8b) — применяются только после вашего подтверждения.
- Главы по смыслу (с приведением к правилам YouTube) и черновик
  YouTube-метаданных (заголовок / описание / теги / хук) в `metadata.txt`.

### Редактор
- Локальный веб-редактор (FastAPI + wavesurfer.js v7, завендорен без CDN):
  волна с цветными регионами вырезов, кликабельный транскрипт, список вырезов,
  preview-montage без рендера, горячие клавиши.
- Ручные вырезы (drag по волне или выделение слов), undo/redo, drag границ
  региона со снэпом по словам, предпросмотр «До/После», редактируемый
  транскрипт (двойной клик — исправить опечатку Whisper).
- Диалог настроек рендера (кодек, качество, караоке-сабы, 9:16, denoise,
  смена моделей Whisper/LLM), live-прогресс (SSE), батч-очередь рендера,
  переживающая рестарт сервера; экспорт в NLE (FCPXML + EDL).

### Рендер и аудио
- Недеструктивный рендер: один кадрово-точный прогон NVENC (constqp 19) или
  x264; оригиналы не трогаются, повторные запуски идемпотентны.
- Цензура мата только по аудио (partial / pitch / lowpass / reverse), видео
  остаётся бит-в-бит; маскирование мата в тексте субтитров.
- Субтитры `.srt`/`.vtt` с лимитами скорости чтения; опционально вшитые
  караоке-сабы (`.ass` / libass, пословная подсветка); `chapters.txt`.
- Denoise (ffmpeg afftdn + highpass; опционально нейро-движок DeepFilterNet 3),
  мастеринг громкости −14 LUFS (1- и 2-pass).

### Clip Maker
- Локальная LLM предлагает кандидатов под Shorts (хук-фраза, 20–60 с, без
  пересечений); рендер выбранных — 9:16 с face-crop (YuNet / OpenCV) по
  диапазону, караоке-сабами и внутренними вырезами.

### Приватность
- 100 % локально: Whisper на вашем GPU, LLM на localhost, ffmpeg — ни видео,
  ни транскрипт не покидают вашу машину; zero-upload бейдж в редакторе.
- Оффлайн-режим `--offline`: блокирует исходящие соединения процесса ко всему,
  кроме localhost.
- Защита от CSRF/DNS-rebinding: Origin/Referer-guard на всех мутирующих
  эндпоинтах + TrustedHost (закреплено 33 тестами).

### Онбординг и надёжность
- Степпер первого запуска (3 шага), подтверждение перед первой загрузкой
  модели Whisper (с размером, ~3,1 ГБ для large-v3), предупреждение при
  фолбэке транскрипции на CPU, карточка «ffmpeg не найден» с кнопкой
  «Проверить снова».
- `GET /api/health`: статус ffmpeg/ffprobe/Ollama и кэша модели Whisper.
- Установка одной командой: `irm …/scripts/install.ps1 | iex`; исправлено
  зависание `run.ps1` при первом запуске через py-лаунчер.
- CI (GitHub Actions): полный набор тестов на Ubuntu и Windows ×
  Python 3.11/3.12 — без ffmpeg/GPU/Ollama (тесты герметичны);
  548 тестов зелёные.
- Опубликован бенчмарк качества резки (`docs/BENCHMARK.md`): 0 % задетых
  слов на 129 авто-вырезах, воспроизводится одной командой.
